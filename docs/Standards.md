## 3. 결과값 상수

| 값 | 의미 | 언제 사용 |
|----|------|----------|
| `Y` | 양호 | 기준 충족 |
| `N` | 취약 | 기준 미충족 |
| `N/A` | 해당없음 | 점검 불가/미해당 |
| `M` | 수동확인 | 자동판단 불가 |
